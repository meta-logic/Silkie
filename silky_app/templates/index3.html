Quick start

Use the scratchpad for a fresh page to write your proofs on. Silky provides basic UI for running and inspecting proofs, similar to IDEs such as CoqIDE, Proof General, and VSCoq.

Actions
ButtonKey bindingAction
F8 Toggles the goal panel.
Alt+↓/↑ or
Alt+N/P 		Move through the proof.
Alt+Enter or
Ctrl+Enter
(⌘ on Mac) 		Run (or go back) to the current point.
Ctrl+ Hover executed statements to peek at the proof state after each step.
